# Privacy Policy

**Last updated: [Date]**

## Data Collection

This extension does **NOT** collect, store, or transmit any user data to external servers.

## Local Storage

All data is stored locally in your browser using Chrome's `chrome.storage.local` API:
- Profile information (names, emails, addresses, etc.)
- Custom fields and file uploads
- Extension settings
- API keys (if you choose to use AI features)

## Third-Party Services

### OpenAI API (Optional)
- If you enable AI classification, your API key is stored locally
- Field data is sent to OpenAI API only when you explicitly use AI features
- We do not have access to your API key or OpenAI responses
- You can disable AI features at any time

## Permissions

- **storage**: Required to save your profiles and settings locally
- **activeTab**: Required to access form fields on the current page
- **scripting**: Required to fill form fields with your data
- **tabs**: Required to open the options page in a new tab
- **<all_urls>**: Required to work on any website with forms

## Data Sharing

We do not share, sell, or transmit your data to any third parties.
